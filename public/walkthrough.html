One-time Setup

Do this once. Takes about 10 minutes.

🖨️

Plug in your Zebra GC420t

Connect the printer to your computer using the USB cable. Make sure it's powered on — the blue power LED should be solid, not blinking.

Load the label roll before starting. The GC420t takes 4" wide rolls. For jewelry tags, use 1.5" × 1" label stock.
📥

Install Zebra Browser Print

This is a free background app from Zebra that lets your browser talk to the printer over USB. Without it, your browser can't send ZPL commands to the printer.

# Download from Zebra's official site:
zebra.com/us/en/software/zebra-browser-print.html

# After installing, it runs in your system tray.
# You'll see a small Zebra Z icon near the clock.
✅ Once it's running, it will auto-detect your GC420t and make it available for browser printing.
🖥️

Make sure Node.js is installed

The system runs on Node.js. Open Terminal and check:

node --version
# Should show v18 or higher, e.g. v25.9.0
If you see "command not found", download Node.js from nodejs.org and install it first.
📂

Install the project once

This downloads all the code dependencies. You only do this once (or if you move the folder).

# Open Terminal and run:
cd ~/Desktop/jeweller-stock
npm install

# You'll see packages downloading. Takes ~30 seconds.

Starting the System Every Day

Two things to open each morning.

▶️

Start the stock server

Open Terminal and run this:

cd ~/Desktop/jeweller-stock
npm start

You'll see:

Jalan Jewellers Stock API running on http://localhost:3000
Press Ctrl+C to stop.
Keep this Terminal window open while you're using the system. Don't close it — closing it stops the server.
🌐

Open the app in your browser

Open Chrome or Safari and go to:

http://localhost:3000

You'll see the Jalan Jewellers dashboard with your stock summary at the top.

✅ Tip: Bookmark this URL so you can open it with one click each morning.

Adding a New Piece of Jewellery

Each item gets its own unique tag number.

➕

Click "Add New Item" tab

At the top of the page, click the Add New Item tab. You'll see a form to fill in the details of the piece.

📝

Fill in the item details

Enter what you know about the piece:

Name — e.g. "Ruby Necklace"
Category — Ring / Necklace / Bangle etc
Metal — Gold / Silver / Platinum
Purity — 22K / 18K / 925 etc
Gross Weight — total weight in grams
Net Weight — metal weight (without stones)
MRP — selling price in ₹
Supplier — who you bought it from
✅

Click Submit

The system saves the item and automatically generates a unique SKU (Stock Keeping Unit number) for it.

# Example SKU format:
JS-20260513-0001
# JS = Jalan Stock
# 20260513 = date added (13 May 2026)
# 0001 = first item added today
✅ The item now appears in your stock table. Next step: print a tag for it.

Printing a Tag for a Piece

This is what gets attached to the jewellery.

What the printed tag looks like

JALAN JEWELLERS
JS-20260513-0001
Gold / 22K / 15.20g
Rs. 75,000

Actual size: 1.5 inches wide × 1 inch tall

🖨️

Click "Print Tag" next to the item

In the stock table, find the item and click the Print Tag button. A popup appears showing:

  • A barcode preview — visual preview of what will print
  • The ZPL code — the raw printer instructions in a text box
📋

Copy the ZPL code

Click the Copy ZPL button. The printer instructions are now on your clipboard.

ZPL (Zebra Programming Language) is a text-based language that tells the Zebra printer exactly what to print and where. You don't need to understand it — just copy and paste it.
🖥️

Open Zebra Browser Print and print

Click the Zebra Z icon in your system tray. The Browser Print interface opens. Then:

  1. Make sure your GC420t is selected in the printer list
  2. Paste the ZPL into the text area (Ctrl+V / Cmd+V)
  3. Click Print
✅ The printer will immediately spit out the tag. Attach it to the jewellery piece using a tag string or adhesive.
💡

Alternative: print directly from Terminal

If you prefer, you can print without opening Browser Print at all. The app shows you the exact command:

lpr -P ZebraGC420t /tmp/label.zpl
# Replace ZebraGC420t with your printer's exact name
# Check printer name with: lpstat -p

Scanning a Tag at the Counter

Point, scan, done — the item pulls up instantly.

No special driver needed — USB barcode scanners act as a keyboard and type the barcode number for you.
🔌

Plug in the barcode scanner

Connect your USB barcode scanner. No driver installation needed — it works like a USB keyboard out of the box.

📍

The scan bar is always ready

When the app loads, the scan bar at the top of the page is automatically focused. You don't need to click anything — just scan and it will type into the box.

If you've been clicking around the page, press Ctrl+F to instantly re-focus the scan bar.
📲

Scan the tag on the jewellery

Point the scanner at the barcode on the tag. It will:

  1. Type the SKU number into the scan bar (JS-20260513-0001)
  2. Automatically press Enter
  3. The item is highlighted and scrolled to in the stock table
✅ You can now see all details for that piece — weight, purity, MRP, who supplied it.
💰

Mark it as Sold (when a customer buys)

After scanning, click Mark Sold on that item. The status changes from:

In Stock → Sold

The item stays in your records — you can always search old sold items for reference.

Day-to-Day Workflow

What a normal day looks like.

🌅 Morning (opening up)

  • Open Terminal → npm start
  • Open Chrome → localhost:3000
  • Check the stats bar — total stock and value
  • Make sure Zebra Browser Print is running

📦 New stock arrives

  • Click Add New Item for each piece
  • Fill in details, click Submit
  • Click Print Tag → copy ZPL → print
  • Attach tag to jewellery piece

👤 Customer is looking at a piece

  • Scan the tag OR type SKU into scan bar
  • Item details appear instantly
  • See purity, weight, MRP at a glance
  • Edit to add a note or reserve it

💰 Customer buys a piece

  • Scan the tag
  • Click Mark Sold
  • Status turns grey → Sold
  • Stock value updates automatically

🔍 Looking up a specific piece

  • Type in the scan bar to search by name
  • Use the filters: category, metal, status
  • Or scan the physical tag if you have it

📊 Checking your stock value

  • Click the Reports tab
  • See total value, by category, by metal
  • Stats bar always shows live totals

What each status means

In Stock Item is in the shop and available for sale
Sold Item has been sold — kept for records but excluded from stock value
Reserved Customer has put it on hold — not sold yet, but not available either
On Repair Sent out for repair or modification — temporarily out of stock

Tips & Common Questions

Things that come up often.

ℹ️
Can I use this on two computers? — Not yet. The database lives on one computer. For multi-device use, the server can be moved to a shared machine on your local network and accessed from any browser via the computer's IP address (e.g. http://192.168.1.10:3000).
⚠️
Where is my data? — All data is in the file ~/Desktop/jeweller-stock/jeweller-stock.db. Back this file up regularly (copy it to a USB drive or Google Drive). If this file is lost, your stock records are gone.
✅
Scanner not reading? — Make sure the scan bar at the top has focus (it turns gold when focused). Press Ctrl+F to focus it. If the scanner still doesn't work, try scanning into Notepad/TextEdit first to confirm it's reading the barcode correctly.
ℹ️
Printer not printing? — Check: (1) Zebra Browser Print is open in the system tray, (2) GC420t is on and connected via USB, (3) label roll is loaded and calibrated. To calibrate: hold the Feed button while powering on until the printer flashes twice.
⚠️
The page shows "Cannot connect"? — The server isn't running. Open Terminal, go to ~/Desktop/jeweller-stock, and run npm start.
✅
Keyboard shortcut: Press Ctrl+F anywhere in the app to jump straight to the scan/search bar. Escape closes any open popup.
